# GT Bailey IT Solutions LLC Landing (GitHub Pages)

This folder is ready to publish on GitHub Pages.

## Structure

- `index.html` - Main landing page
- `assets/css/styles.css` - Site styles
- `assets/js/main.js` - Optional JavaScript file
- `assets/images/` - Local image folder (optional)
- `.nojekyll` - Prevents Jekyll processing on GitHub Pages
- `CNAME.template` - Example custom-domain file

## Publish Steps

1. Create a new GitHub repo.
2. Copy these files into the repo root.
3. Commit and push to `main`.
4. In GitHub: `Settings -> Pages -> Deploy from branch -> main /(root)`.
5. Optional: for a custom domain, rename `CNAME.template` to `CNAME` and set your domain inside it.
